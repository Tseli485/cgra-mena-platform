🛡️ Plateforme MENA Tuteur

Application de bureau — aucune installation, double-clic et c'est parti · Bureaubladapplicatie — geen installatie nodig

🌐 Le plus simple : utilisez directement la version web — ouvrir l'application. Les téléchargements ci-dessous servent à travailler sans navigateur configuré ou à transmettre l'app à un collègue.

🪟 Windows

Double-cliquez sur le fichier. Si SmartScreen s'affiche : « Informations complémentaires » → « Exécuter quand même ».

Télécharger .exe

ou via winget :
winget install MENA-Tuteur

🍎 macOS

Dézippez puis clic droit → Ouvrir (1re fois uniquement, app non signée).

Télécharger .app.zip

🐧 Linux

chmod +x MENA-Tuteur-Linux puis exécutez. Python non requis.

Télécharger
🔒 Vérifier l'intégrité : comparez le SHA-256 de votre fichier avec SHA256SUMS.txt (PowerShell : Get-FileHash MENA-Tuteur-Windows.exe).
📱 Sur mobile/tablette : ouvrez la version web puis menu du navigateur → « Installer l'application ».
