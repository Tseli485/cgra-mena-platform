# Checklist Détaillée du Tuteur / Guardian Detailed Checklist

## Daily Responsibilities Checklist

### Morning (Start of Day)
- [ ] Child health check: Physical condition, mood, visible injuries
- [ ] Communication review: Any messages from school, parents, social services
- [ ] Medication verification: All required medications in place, doses tracked
- [ ] Schedule coordination: Confirm appointments, activities, logistics
- [ ] Safety inspection: Hazard assessment of environment

### During Supervision
- [ ] Active engagement: Focused attention on child's activities
- [ ] Observable supervision: Maintaining visual/audio contact
- [ ] Incident documentation: Any concerning behaviors noted
- [ ] Communication: Clear instructions, responsive to questions
- [ ] Boundaries maintenance: Professional demeanor maintained

### End of Day
- [ ] Health observation: Any physical/emotional changes noted
- [ ] Progress review: Activities completed, learning observed
- [ ] Preparation: Next day's materials, notifications sent
- [ ] Documentation: Session notes completed
- [ ] Environment check: Safety for next session

---

## Weekly Responsibilities Checklist

### Educational Progress
- [ ] Learning goals assessment: Progress toward agreed targets
- [ ] Communication with school: Teacher feedback collected
- [ ] Materials review: Books, supplies, resources appropriate
- [ ] Skill development: New concepts introduced, progress tracked
- [ ] Parent consultation: Weekly report prepared or delivered

### Child Wellbeing
- [ ] Emotional check-in: How is child feeling about tutoring?
- [ ] Physical health: Any medical concerns, wellness observed
- [ ] Family dynamics: Any changes in home situation
- [ ] Peer relationships: Social interaction observations
- [ ] Behavioral patterns: Any concerning trends noted

### Administrative
- [ ] Record keeping: All notes and observations recorded
- [ ] Payment/contract: Hours logged, payment processed
- [ ] Communication review: Messages responded to
- [ ] Planning: Next week's sessions prepared
- [ ] Equipment check: Materials, supplies ready

---

## Monthly Responsibilities Checklist

### Assessment & Reporting
- [ ] Progress evaluation: Formal assessment of learning
- [ ] Goal adjustment: Targets reviewed, modified if needed
- [ ] Parent meeting: In-person or formal written report
- [ ] School consultation: Report to school (if applicable)
- [ ] Documentation: Monthly summary completed

### Professional Development
- [ ] Training review: Any new skills/knowledge acquired
- [ ] Best practices update: Latest methods reviewed
- [ ] Continuing education: Hours logged toward requirements
- [ ] Supervision: Meeting with supervisor/manager
- [ ] Self-reflection: Strengths and areas for improvement noted

### Child Development
- [ ] Developmental assessment: Age-appropriate progress confirmed
- [ ] Social skills: Peer relationships, cooperation
- [ ] Emotional growth: Resilience, self-esteem development
- [ ] Academic trajectory: Subject competency evaluation
- [ ] Health review: Medical appointments current

### Compliance
- [ ] Background clearance: Status verified (if periodic renewal required)
- [ ] Insurance: Professional liability current
- [ ] Tax/legal: Compliance documentation current
- [ ] Reporting obligations: Child safety concerns reported if applicable
- [ ] Documentation: All records properly maintained

---

## Quarterly Responsibilities Checklist

### Comprehensive Review
- [ ] Overall progress: 3-month goal achievement evaluation
- [ ] Child development: Formal assessment results reviewed
- [ ] Family satisfaction: Feedback collected from parents
- [ ] School feedback: Teacher input on academic/behavioral progress
- [ ] Goal setting: New targets for next quarter

### Training & Compliance
- [ ] Certification status: All required training current
- [ ] Continuing education: Minimum hours completed
- [ ] Incident review: Any concerns, how handled, outcomes
- [ ] Professional development: Plan for next quarter
- [ ] Supervision reflection: Regular check-in with manager

### Documentation Audit
- [ ] Records completeness: All sessions documented
- [ ] Confidentiality: Secure storage, access controlled
- [ ] Child safety concerns: Properly reported and followed up
- [ ] Communication: All parent/school correspondence documented
- [ ] Financial: Hours and payment records accurate

---

## Annual Responsibilities Checklist

### Comprehensive Assessment
- [ ] Annual learning goal: Full-year progress evaluated
- [ ] Developmental milestones: Age-appropriate achievement confirmed
- [ ] Family relationship: Quality of partnership with parents
- [ ] Professional growth: Skills development, training completed
- [ ] Child safety: Zero incidents of concern or all properly reported

### Formal Review & Renewal
- [ ] Performance review: Supervisor meeting, feedback received
- [ ] Background clearance renewal: Application/verification submitted
- [ ] Insurance renewal: Liability coverage confirmed
- [ ] Tax compliance: Annual return/documentation submitted
- [ ] Training renewal: All certifications current for next year

### Future Planning
- [ ] Continuation: Decision on ongoing engagement made
- [ ] Goal setting: Annual objectives for next year
- [ ] Family consultation: Long-term planning with parents
- [ ] Professional development: Training/education plan for year
- [ ] Documentation: Annual summary report completed

---

## Emergency/Incident Checklist

### Immediate Response (Child Safety Threat)
- [ ] Secure child safety: Immediate protection measures taken
- [ ] Emergency services: Called if immediate medical/police need
- [ ] Incident documentation: Exact details recorded
- [ ] Parent notification: Family informed immediately
- [ ] Supervisor notification: Manager/supervisor contacted
- [ ] Child support: Emotional support provided as appropriate

### Follow-Up (Within 24 Hours)
- [ ] Mandatory reporting: Child protective services contacted if abuse suspected
- [ ] Medical attention: Any injuries assessed/treated
- [ ] Investigation cooperation: Full transparency with authorities
- [ ] Record keeping: Detailed incident report filed
- [ ] Communication: All stakeholders updated

### Ongoing Support
- [ ] Child wellbeing: Trauma-informed care provided
- [ ] Family support: Parents offered resources/guidance
- [ ] Professional support: Supervisor/counselor consulted for guidance
- [ ] Documentation: All follow-up recorded
- [ ] Prevention: Measures taken to prevent recurrence
