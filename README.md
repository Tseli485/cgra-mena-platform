# MENA Tuteur Training Platform v1.0.0

**Comprehensive digital training and reference system for child guardianship tuteurs in Belgium**

## 🚀 Quick Start

### 🌐 **Live Web App**
https://cgra-mena-platform.netlify.app

### 📚 **Key Resources**
- **20-page Quick Reference:** Download SYLLABUS_COURTE.pdf
- **80-page Workbook:** Download SYLLABUS_COMPLETE.pdf
- **Resource Dossiers:** Access in-app

### 📱 **Install as App**
Open the web app in your browser and select "Install" or "Add to Home Screen"

### 📖 Complete Curriculum
- **Lifecycle Guidance** (Ages 0-18)
  - Prenatal to early childhood (0-3)

  - Family dynamics, abuse, behavioral issues
  - Mental health, school issues, trauma
  - Specialized: Trafficking, LGBTQ+, disability

  - Starting tutorship

- Professional forms and templates
- Sample letters and notices

## 📖 How to Use

### For Tuteurs
1. Read quick reference guide (20 min)
2. Review lifecycle section for your cases (30 min)
3. Search for specific procedures needed
4. Reference case studies for examples
5. Use resource directory for contacts

### For Organizations
1. Install app on team devices
2. Distribute printed guides
3. Use in training programs
4. Reference in supervision
5. Share with staff

### For Training Programs
1. Include in curriculum
2. Reference case studies in lessons
3. Use resource directory for referrals
4. Practice with forms and templates
5. Discuss with trainees
