# Checklists by Age Module
Guardianship Action Checklists for Each Life Stage

---

## Age 0-2 Years: Foundation & Registration Checklist

**Priority Level:** IMMEDIATE (Days 1-30) | ONGOING (Months 1-24)

### Immediate Actions (Within 30 Days)
- [ ] **Register guardianship decision** with local judicial authority
- [ ] **Obtain birth certificate** (if not already available)
- [ ] **Schedule initial medical examination** with pediatrician
- [ ] **Register child with health insurance** (national or private)
- [ ] **Update family contact information** with authorities
- [ ] **Establish dedicated bank account** for child's funds
- [ ] **Create document filing system** (physical + digital backup)
- [ ] **Notify previous caregivers** of guardianship change

### First 3 Months Actions
- [ ] **Establish regular sleep/feeding schedule** documentation
- [ ] **Complete initial vaccination schedule** verification
- [ ] **Register for childcare if needed** (kindergarten/daycare)
- [ ] **Schedule development assessment** (12-week check)
- [ ] **Photograph documentation** (health, living conditions)
- [ ] **Open family communication log** (visits, contacts)
- [ ] **Complete guardianship training** (if required by jurisdiction)
- [ ] **Set up monthly financial tracking** system

### Ongoing Actions (Months 3-24)
- [ ] **Monthly medical check-ups** (follow-up appointments)
- [ ] **Vaccination schedule on track** (update records after each visit)
- [ ] **Nutrition/feeding documentation** (growth monitoring)
- [ ] **Monthly budget reconciliation** (food, healthcare, clothing)
- [ ] **Family visits scheduled** (minimum monthly if applicable)
- [ ] **Development milestone tracking** (rolling updates)
- [ ] **Emergency contact information current** (update quarterly)
- [ ] **Annual medical report** prepared for authorities
- [ ] **Living conditions inspected** (cleanliness, safety, temperature)
- [ ] **Educational foundation beginning** (exposure, language development)

### Documentation to Gather
- [ ] Birth certificate (certified copy)
- [ ] Medical examination report
- [ ] Vaccination record card
- [ ] Health insurance documentation
- [ ] Bank account statements (monthly)
- [ ] Receipt folder (organized by category)
- [ ] Family contact log
- [ ] Guardianship decision copy

**Timeline Indicator:** Monthly reviews | Quarterly authority updates

- See *Obligations_Summary.md* > Financial Obligations for budget allocation
- See *Role_Tuteur.md* > Caregiving Responsibilities for daily routines
- See *MENA_Definition.md* > Legal Framework for registration requirements

## Age 2-6 Years: School Preparation & Integration Checklist

**Priority Level:** IMMEDIATE (Months 1-2) | ONGOING (Months 3-48)

### Pre-School Enrollment (Before Age 3)
- [ ] **Assess educational options** (public school, private, special needs programs)
- [ ] **Complete school enrollment process** (get acceptance letter)
- [ ] **Provide required documents to school** (birth certificate, vaccination proof, ID)
- [ ] **Schedule school orientation visit** (introduction to environment)
- [ ] **Arrange transportation** (public transit, school bus, or personal)
- [ ] **Confirm school fee structure** (budget adjustment if needed)
- [ ] **Discuss any special needs** with school administration

### Health & Development (Ages 2-6)
- [ ] **Complete required vaccinations** (pre-school mandate verification)
- [ ] **Schedule annual dental check-up** (first visit by age 3)
- [ ] **Conduct developmental screening** (speech, motor, cognitive)
- [ ] **Obtain vision/hearing assessment** (baseline at age 3-4)
- [ ] **Update growth charts monthly** (track nutrition adequacy)
- [ ] **Address any health concerns immediately** (infections, allergies, disabilities)
- [ ] **Maintain medication schedule** (if applicable)
- [ ] **Document special dietary needs** (allergies, preferences)

### Social & Behavioral Integration
- [ ] **Monitor peer interactions** (observe play patterns, social skills)
- [ ] **Set behavioral expectations** (age-appropriate rules)
- [ ] **Establish daily routine** (consistent wake, meals, bedtime)
- [ ] **Introduce emotional regulation** (calming strategies, naming feelings)
- [ ] **Record behavioral incidents** (tantrums, aggression, fears)
- [ ] **Schedule behavioral assessment** (if concerns arise)
- [ ] **Facilitate family relationships** (visits, phone calls as appropriate)
- [ ] **Document language development** (milestones, language exposure)

### Educational Foundation
- [ ] **Support language learning** (exposure to school language if different from home)
- [ ] **Provide learning materials** (age-appropriate books, toys, activities)
- [ ] **Establish reading routine** (daily 15-20 minutes minimum)
- [ ] **Encourage artistic expression** (drawing, painting, music)
- [ ] **Track school performance** (attendance, teacher feedback, progress)
- [ ] **Arrange tutoring if needed** (early intervention for delays)
- [ ] **Update school records** (any changes in contact, health, emergencies)

### Financial Management (Ages 2-6)
- [ ] **Monthly food budget tracking** (childcare nutrition needs increase)
- [ ] **School fee payments on schedule** (no arrears)
- [ ] **Education materials budget** (books, supplies, uniforms)
- [ ] **Health expense documentation** (dental, vision, specialist visits)
- [ ] **Clothing budget adjustment** (growth, replacement needs)
- [ ] **Activity fee tracking** (sports, arts, enrichment programs)
- [ ] **Quarterly financial reconciliation** with authority
- [ ] **Reserve fund maintained** (emergency cushion updated)

### Ongoing Monitoring
- [ ] **School progress reviews** (quarterly teacher meetings)
- [ ] **Medical appointments kept** (annual + as-needed visits)
- [ ] **Family contact maintained** (monthly minimum)
- [ ] **Incident log updated** (behavioral, health, educational concerns)
- [ ] **Authority reports submitted** (annual or as required)
- [ ] **Document storage organized** (files easy to locate)

**Timeline Indicator:** Monthly family contact | Quarterly school reviews | Annual authority report

**Cross-References:**
- See *Role_Tuteur.md* > Educational Support for school integration guidance
- See *Obligations_Summary.md* > Document Requirements for school documentation
- See *MENA_Definition.md* > Child Rights for educational rights framework

## Age 6-12 Years: Education & Development Checklist

**Priority Level:** IMMEDIATE (Month 1) | ONGOING (Months 2-132)

### Educational Planning & Tracking
- [ ] **Confirm school enrollment** (public/private primary school)
- [ ] **Introduce academic support** (tutoring if needed, reading clubs)
- [ ] **Obtain academic record** (baseline assessment, previous school reports)
- [ ] **Establish homework routine** (scheduled study time, parent oversight)
- [ ] **Monitor academic progress** (grades, test scores, teacher feedback)
- [ ] **Schedule parent-teacher meetings** (minimum 2 per year)
- [ ] **Address learning gaps** (intervention plan if struggling)
- [ ] **Identify strengths/interests** (aptitude assessment by age 8-9)
- [ ] **Prepare for secondary school transition** (by age 11-12)
- [ ] **Track attendance** (absence explanation, truancy prevention)

### Health & Physical Development
- [ ] **Annual physical examination** (comprehensive health check)
- [ ] **Dental care on schedule** (cleaning, orthodontics assessment)
- [ ] **Vision & hearing screening** (pre-school, mid-childhood)
- [ ] **Height/weight monitoring** (growth trajectory tracking)
- [ ] **Address puberty education** (age-appropriate by age 10-11)
- [ ] **Sexual abuse prevention training** (age-appropriate education)
- [ ] **Accident/injury documentation** (school or home incidents)
- [ ] **Mental health assessment** (counseling if behavioral concerns)
- [ ] **Maintain vaccination updates** (booster shots as needed)
- [ ] **Medication compliance** (if applicable, documentation)

### Social & Emotional Development
- [ ] **Monitor friendships** (peer relationships, social inclusion)
- [ ] **Support activity participation** (sports, clubs, arts)
- [ ] **Teach conflict resolution skills** (age-appropriate strategies)
- [ ] **Maintain family connections** (monthly visits/calls minimum)
- [ ] **Document significant milestones** (achievements, challenges)
- [ ] **Address behavioral issues** (bullying, aggression, withdrawal)
- [ ] **Provide emotional support** (counseling if trauma-related)
- [ ] **Encourage cultural identity** (heritage activities, celebrations)
- [ ] **Foster independence** (age-appropriate responsibilities)

### Legal & Administrative Status
- [ ] **Update identification documents** (if expired, renew)
- [ ] **Guardianship status confirmed** (annually with authorities)
- [ ] **File annual legal report** (progress, needs, challenges)
- [ ] **Maintain communication with authorities** (respond to inquiries)
- [ ] **Address changes in circumstance** (new living situation, family status)
- [ ] **Consent forms current** (medical, school, travel authorizations)
- [ ] **Insurance coverage verified** (health, educational liability)

### Life Skills Foundation
- [ ] **Age 6-8: Basic self-care** (dressing, hygiene, eating)
- [ ] **Age 8-10: Household tasks** (chores, organization, responsibility)
- [ ] **Age 10-12: Safety skills** (pedestrian, internet, emergency contacts)
- [ ] **Age 11-12: Money basics** (allowance, saving, simple budgeting)
- [ ] **Introduce vocational interests** (field trips, mentor exposure)
- [ ] **Develop decision-making skills** (age-appropriate choices)

### Financial Responsibility (Ages 6-12)
- [ ] **School fee payments current** (no arrears)
- [ ] **Education materials provided** (books, uniforms, supplies)
- [ ] **Extracurricular activities budgeted** (sports, music, clubs)
- [ ] **Health/medical expenses tracked** (dental, vision, therapy)
- [ ] **Clothing budget adjusted** (growth, seasonal replacement)
- [ ] **Monthly budget reconciliation** (expenses vs. allocations)
- [ ] **Child savings account maintained** (birthday gifts, earned money)
- [ ] **Quarterly authority financial reports** submitted

### Ongoing Monitoring
- [ ] **School performance reviewed** (quarterly at minimum)
- [ ] **Health appointments maintained** (annual exams + as-needed)
- [ ] **Family relationships tracked** (contact log, visit frequency)
- [ ] **Incident documentation** (academic, behavioral, health-related)
- [ ] **Authority inspection cooperation** (documents ready, access available)
- [ ] **Document files organized** (easy retrieval, backup current)

**Timeline Indicator:** Monthly family contact | Quarterly school meetings | Annual authority report | Bi-annual physical exams

- See *Obligations_Summary.md* > Timeline Overview for age milestone requirements

## Age 12-18 Years: Independence & Transition Checklist

**Priority Level:** IMMEDIATE (Month 1) | ONGOING (Months 2-72) | CRITICAL (Age 17-18)

### Secondary Education & Career Planning
- [ ] **Enroll in secondary school** (confirm placement, obtain acceptance)
- [ ] **Review academic transcript** (baseline performance assessment)
- [ ] **Establish academic support** (tutoring, mentoring if needed)
- [ ] **Identify career interests** (vocational assessment by age 14)
- [ ] **Explore vocational options** (internships, apprenticeships by age 15)
- [ ] **Monitor school attendance** (truancy prevention, engagement)
- [ ] **Maintain strong academic performance** (grades, test preparation)
- [ ] **Plan post-secondary pathway** (university, vocational, employment)
- [ ] **Secure necessary certifications** (driver's license, work permit where applicable)
- [ ] **Prepare for school transitions** (new school entry, graduation planning)

### Work & Economic Independence (Age 14+)
- [ ] **Obtain work permit** (if jurisdiction-dependent, apply before age 15-16)
- [ ] **Arrange safe employment** (age-appropriate, authorized employers)
- [ ] **Monitor work hours** (compliance with labor laws, school balance)
- [ ] **Teach workplace skills** (punctuality, professionalism, safety)
- [ ] **Track earnings** (income documentation, tax considerations)
- [ ] **Develop financial literacy** (budgeting, saving for future)
- [ ] **Build employment record** (references, experience documentation)
- [ ] **Plan for economic independence** (post-18 employment strategy)

### Health & Physical Development
- [ ] **Annual health examination** (comprehensive check-up)
- [ ] **Sexual health education** (age-appropriate, preventive care)
- [ ] **Mental health support** (counseling, therapy if needed)
- [ ] **Dental care maintained** (regular cleaning, orthodontics completion)
- [ ] **Vision/hearing screening** (update as needed)
- [ ] **Vaccine booster schedule** (tetanus, HPV, meningitis where applicable)
- [ ] **Reproductive health** (contraception education if applicable)
- [ ] **Address chronic conditions** (management plan, medication compliance)
- [ ] **Substance abuse prevention** (education, monitoring for warning signs)
- [ ] **Document health baseline** (for future medical history)

### Social & Emotional Maturity
- [ ] **Support peer relationships** (healthy friendships, social inclusion)
- [ ] **Address behavioral issues** (aggression, risk-taking, withdrawal)
- [ ] **Provide mental health support** (counseling for trauma, depression, anxiety)
- [ ] **Maintain family connections** (frequent contact, relationship building)
- [ ] **Prepare for separation anxiety** (when guardianship ends)
- [ ] **Encourage identity development** (cultural, personal values, beliefs)
- [ ] **Foster responsibility** (decision-making, consequence understanding)
- [ ] **Build resilience** (coping skills, problem-solving strategies)

### Life Skills & Independence Training
- [ ] **Age 12-14: Basic cooking** (simple meals, nutrition awareness)
- [ ] **Age 12-14: Household management** (laundry, cleaning, organization)
- [ ] **Age 14-16: Finance management** (banking, budgeting, savings)
- [ ] **Age 14-16: Health self-care** (appointments, medication management)
- [ ] **Age 16-18: Emergency response** (first aid, crisis management)
- [ ] **Age 16-18: Legal awareness** (rights, responsibilities, contracts)
- [ ] **Age 17-18: Housing skills** (apartment hunting, lease understanding)
- [ ] **Age 17-18: Employment skills** (resume, interviews, workplace navigation)
- [ ] **Age 17-18: Community navigation** (public services, transportation, resources)

### Transition Planning (Age 16-18)
- [ ] **Conduct comprehensive life skills assessment** (age 16)
- [ ] **Develop independence transition plan** (with youth input, age 16-17)
- [ ] **Identify post-guardianship support** (housing, education, employment, mental health)
- [ ] **Connect with adult services** (social services, vocational programs, housing assistance)
- [ ] **Establish financial security plan** (savings account, emergency fund, earned income)
- [ ] **Update legal documents** (emancipation paperwork, power of attorney if applicable)
- [ ] **Finalize educational/vocational pathway** (college application, job placement, apprenticeship)
- [ ] **Prepare exit strategy** (gradual independence, mentorship continuation)
- [ ] **Document achievements** (certificates, skills, accomplishments)

### Legal & Administrative Status
- [ ] **Confirm guardianship status** (annual update with authorities)
- [ ] **File annual legal reports** (progress, challenges, transition planning)
- [ ] **Address document renewals** (ID card, passport, work permit)
- [ ] **Maintain communication with authorities** (respond to inquiries, attend meetings)
- [ ] **Update emergency contacts** (current and verified annually)
- [ ] **Understand rights & obligations** (legal literacy, age of majority awareness)
- [ ] **Prepare for guardianship termination** (age 18 or earlier if applicable)
- [ ] **Establish post-guardianship relationship** (mentorship, ongoing support options)

### Financial Responsibility (Ages 12-18)
- [ ] **School costs covered** (tuition, materials, activities, transportation)
- [ ] **Earned income tracked** (work permit jobs, allowance management)
- [ ] **Youth savings account maintained** (for independence fund)
- [ ] **Budget reconciliation quarterly** (authority reporting)
- [ ] **Cost-of-living education** (rent, utilities, food, transportation estimates)
- [ ] **Vocational training costs budgeted** (apprenticeship fees if applicable)
- [ ] **Health/medical expenses documented** (specialized care, therapy)
- [ ] **Emergency fund established** (for post-18 independence)

### Ongoing Monitoring & Documentation
- [ ] **School performance tracked** (grades, attendance, behavior)
- [ ] **Health appointments kept** (annual exams + specialist visits)
- [ ] **Family contact maintained** (monthly minimum, increased frequency age 17+)
- [ ] **Work/employment documented** (hours, earnings, supervisors)
- [ ] **Achievement recognition** (academic, athletic, artistic accomplishments)
- [ ] **Incident documentation** (behavioral, health, disciplinary issues)
- [ ] **Authority inspection cooperation** (full document access, transparent reporting)
- [ ] **Transition readiness assessment** (age 17-18, formal evaluation)

**Timeline Indicator:** Monthly family contact | Quarterly school reviews | Annual health exams | Annual authority reports | Intensive monitoring age 17-18

**Critical Dates (Age 17-18):**
- [ ] Emancipation paperwork initiated (6 months before age of majority)
- [ ] Housing plan finalized (3 months before guardianship ends)
- [ ] Employment/education confirmation (2 months before)
- [ ] Financial handoff process begun (1 month before)
- [ ] Final authority report submitted (upon guardianship termination)
